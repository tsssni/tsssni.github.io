иғҪйҮҸ
#

е…үжәҗеҸ‘еҮәе…үеӯҗ, е…үеӯҗжҗәеёҰиғҪйҮҸ. иғҪйҮҸеҚ•дҪҚдёәз„ҰиҖі(J), зү№е®ҡжіўй•ҝзҡ„е…үеӯҗжҗәеёҰзҡ„иғҪйҮҸеҰӮдёӢејҸ. е…¶дёӯcдёәе…үйҖҹ, hдёәPlanckеёёйҮҸ, h=6.626Г—10вҲ’34m2kg/sh = 6.626\times 10^{-34} m^2 kg/sh=6.626Г—10вҲ’34m2kg/s.

иҫҗе°„йҖҡйҮҸ
#

иҫҗе°„йҖҡйҮҸжҳҜ(radiant flux)еҚ•дҪҚж—¶й—ҙйҖҡиҝҮиЎЁйқўжҲ–з©әй—ҙзҡ„иғҪйҮҸ, д№ҹеҸҜд»Ҙиў«з§°дҪңеҠҹзҺҮ(power), еҚ•дҪҚдёәJ/s, еҚіз“Ұзү№(W). иҫҗе°„йҖҡйҮҸзҡ„е®ҡд№үи§ҒдёӢејҸ.

ОҰ=dQdt \begin{equation} \Phi = \frac{\mathrm{d}Q}{\mathrm{d}t} \end{equation} ОҰ=dtdQвҖӢвҖӢвҖӢ

еҜ№дәҺдёҖдёӘзӮ№е…үжәҗ, иҫҗе°„йҖҡйҮҸдёәд»Ҙе…үжәҗдёәзҗғеҝғзҡ„зҗғйқўеҚ•дҪҚж—¶й—ҙжҺҘж”¶еҲ°зҡ„иғҪйҮҸ, зҗғзҡ„еҚҠеҫ„дёҚеҪұе“Қиҫҗе°„йҖҡйҮҸзҡ„еҖј.

иҫҗе°„з…§еәҰ & иҫҗе°„еҮәе°„еәҰ
#

иҫҗе°„з…§еәҰ(irradiance)жҳҜеҚ•дҪҚйқўз§ҜжҺҘж”¶еҲ°зҡ„иҫҗе°„йҖҡйҮҸ, дёҺд№ӢеҜ№еә”зҡ„иҫҗе°„еҮәе°„еәҰ(radiant exitance)жҳҜеҚ•дҪҚйқўз§ҜеҸ‘еҮәзҡ„иҫҗе°„йҖҡйҮҸ, д»–д»¬зҡ„еҚ•дҪҚдёәW/m2W/m^2W/m2.

E(p)=dОҰdA \begin{equation} E(p) = \frac{\mathrm{d}\Phi}{\mathrm{d}A} \end{equation} E(p)=dAdОҰвҖӢвҖӢвҖӢ

еҗҢж ·д»ҘзӮ№е…үжәҗдёәдҫӢ, д»Ҙе…үжәҗдёәзҗғеҝғзҡ„зҗғйқўзҡ„иҫҗе°„з…§еәҰи§ҒдёӢејҸ.

E=ОҰ4ПҖr2 \begin{equation} E = \frac{\Phi}{4 \pi r^2} \end{equation} E=4ПҖr2ОҰвҖӢвҖӢвҖӢ

йқһеһӮзӣҙе…үзәҝж №жҚ®жҠ•еҪұеҲ°е…үзәҝеһӮзӣҙйқўзҡ„йқўз§Ҝи®Ўз®—, Оё\thetaОёдёәе…үзәҝдёҺжі•зәҝзҡ„еӨ№и§’.

E(p)=dОҰcosвҒЎОёdA \begin{equation} E(p) = \frac{\mathrm{d}\Phi \cos\theta}{\mathrm{d}A} \end{equation} E(p)=dAdОҰcosОёвҖӢвҖӢвҖӢ

иҫҗе°„ејәеәҰ
#

иҫҗе°„ејәеәҰ(radiant intensity)жҳҜеҚ•дҪҚз«ӢдҪ“и§’дёҠзҡ„иҫҗе°„йҖҡйҮҸ, е®ғжҸҸиҝ°дәҶе…үеңЁж–№еҗ‘дёҠзҡ„еҲҶеёғ, дҪҶе®һйҷ…дёҠеҸӘеҜ№зӮ№е…үжәҗжңүж„Ҹд№ү.

I=dQdПү \begin{equation} I = \frac{\mathrm{d}Q}{\mathrm{d}\omega} \end{equation} I=dПүdQвҖӢвҖӢвҖӢ

д»ҘзӮ№е…үжәҗдёәзҗғеҝғзҡ„зҗғйқўзҡ„иҫҗе°„ејәеәҰи§ҒдёӢејҸ.

I=ОҰ4ПҖ \begin{equation} I = \frac{\Phi}{4 \pi} \end{equation} I=4ПҖОҰвҖӢвҖӢвҖӢ

иҫҗе°„дә®еәҰ
#

иҫҗе°„дә®еәҰ(radiance)жҳҜеңЁеҚ•дҪҚйқўз§Ҝзҡ„еҚ•дҪҚз«ӢдҪ“и§’дёҠзҡ„иҫҗе°„йҖҡйҮҸ, еҚіе°Ҷиҫҗе°„з…§еәҰиҝӣдёҖжӯҘеңЁжҜҸдёӘж–№еҗ‘дёҠеҫ®еҲҶ. иҫҗе°„дә®еәҰе®ҡд№үи§ҒдёӢејҸ, е…¶дёӯEПүE_{\omega}EПүвҖӢд»ЈиЎЁдёҺПү\omegaПүеһӮзӣҙзҡ„иЎЁйқўзҡ„иҫҗе°„з…§еәҰ.

L(p,Пү)=EПү(p)dПү=d2ОҰdПүdAвҠҘ=d2ОҰdПүdAcosвҒЎОё \begin{equation} L(p, \omega) = \frac{E_{\omega}(p)}{\mathrm{d}\omega} = \frac{d^2\Phi}{\mathrm{d}\omega \mathrm{d}A^{\perp}} = \frac{d^2\Phi}{\mathrm{d}\omega \mathrm{d}A \cos\theta} \end{equation} L(p,Пү)=dПүEПүвҖӢ(p)вҖӢ=dПүdAвҠҘd2ОҰвҖӢ=dПүdAcosОёd2ОҰвҖӢвҖӢвҖӢ

з”ұдәҺйҡҸзқҖдј ж’ӯиҢғеӣҙеўһеӨ§, еҚ•дҪҚйқўз§Ҝзҡ„иҫҗе°„йҖҡйҮҸеҮҸе°Ҹ, дҪҶеҗҢж—¶е®ғжүҖеҚ зҡ„з«ӢдҪ“и§’д№ҹеҮҸе°Ҹ, еӣ жӯӨе…үеңЁзңҹз©әдёӯдј ж’ӯж—¶е…·жңүиҫҗе°„дә®еәҰдёҚеҸҳзҡ„зү№жҖ§.

е…Ҙе°„/еҮәе°„иҫҗе°„ж–№зЁӢ
#

иҫҗе°„дә®еәҰеңЁиЎЁйқўиҫ№з•ҢеӨ„жңӘеҝ…жҳҜиҝһз»ӯзҡ„, иЎЁйқўжҹҗзӮ№дёҠж–№дёҺдёӢж–№зҡ„жһҒйҷҗеҖјйңҖиҰҒеҚ•зӢ¬иЎЁзӨә.

L+(p,Пү)=limвҒЎtвҶ’0+L(p+tnp,Пү) \begin{equation} L^+(p,\omega)=\lim_{t \to 0^+} L(p + t \mathbf{n}_p, \omega) \end{equation} L+(p,Пү)=tвҶ’0+limвҖӢL(p+tnpвҖӢ,Пү)вҖӢвҖӢLвҲ’(p,Пү)=limвҒЎtвҶ’0вҲ’L(pвҲ’tnp,Пү) \begin{equation} L^-(p,\omega)=\lim_{t \to 0^-} L(p - t \mathbf{n}_p, \omega) \end{equation} LвҲ’(p,Пү)=tвҶ’0вҲ’limвҖӢL(pвҲ’tnpвҖӢ,Пү)вҖӢвҖӢ

дёәи§ЈеҶіиҝҷдёҖй—®йўҳйңҖиҰҒеҢәеҲҶе…Ҙе°„дёҺеҮәе°„иҫҗе°„дә®еәҰ.

Li(p,Пү)={L+(p,вҲ’Пү),ПүвӢ…np>0LвҲ’(p,вҲ’Пү),ПүвӢ…np<0 \begin{equation} L_i(p, \omega) = \begin{cases} L^+(p, -\omega), & \omega \cdot \mathbf{n}_p > 0\\ L^-(p, -\omega), & \omega \cdot \mathbf{n}_p < 0 \end{cases} \end{equation} LiвҖӢ(p,Пү)={L+(p,вҲ’Пү),LвҲ’(p,вҲ’Пү),вҖӢПүвӢ…npвҖӢ>0ПүвӢ…npвҖӢ<0вҖӢвҖӢвҖӢLi(p,Пү)={L+(p,Пү),ПүвӢ…np>0LвҲ’(p,Пү),ПүвӢ…np<0 \begin{equation} L_i(p, \omega) = \begin{cases} L^+(p, \omega), & \omega \cdot \mathbf{n}_p > 0\\ L^-(p, \omega), & \omega \cdot \mathbf{n}_p < 0 \end{cases} \end{equation} LiвҖӢ(p,Пү)={L+(p,Пү),LвҲ’(p,Пү),вҖӢПүвӢ…npвҖӢ>0ПүвӢ…npвҖӢ<0вҖӢвҖӢвҖӢ

еҜ№дәҺз©әй—ҙдёӯдёҚеңЁиЎЁйқўдёҠзҡ„жҹҗзӮ№, иҫҗе°„дә®еәҰжҳҜиҝһз»ӯзҡ„.

Lo(p,Пү)=Li(p,вҲ’Пү)=L(p,Пү) \begin{equation} L_o(p, \omega) = L_i(p, -\omega) = L(p, \omega) \end{equation} LoвҖӢ(p,Пү)=LiвҖӢ(p,вҲ’Пү)=L(p,Пү)вҖӢвҖӢ

иҫҗе°„е…үи°ұеҲҶеёғ
#

иҫҗе°„дә®еәҰеҸҜд»ҘиҝӣдёҖжӯҘеҜ№жіўй•ҝеҫ®еҲҶд»ҘиҺ·еҸ–иҫҗе°„е…үи°ұеҲҶеёғ.

LО»=dLdО» \begin{equation} L_\lambda = \frac{\mathrm{d}L}{\mathrm{d}\lambda} \end{equation} LО»вҖӢ=dО»dLвҖӢвҖӢвҖӢ

дә®еәҰдёҺе…үеәҰ
#

иҫҗе°„еәҰеҚ•дҪҚйғҪжңүдёҺд№ӢеҜ№еә”зҡ„е…үеәҰеҚ•дҪҚ, иҝҷйҮҢеҸӘи®Ёи®әжңҖеҹәжң¬зҡ„дә®еәҰ, е…¶дҪҷеҸҜд»Ҙз§ҜеҲҶеҫ—еҲ°. дә®еәҰиЎЁзӨәдәәзңји§ӮеҜҹеҲ°зҡ„жҹҗдёӘе…үи°ұеҠҹзҺҮеҲҶеёғзҡ„еҖј, е®ҡд№үеҰӮдёӢ. Vдёәе“Қеә”жӣІзәҝеҚідәәзңјеҜ№еҗ„дёӘжіўй•ҝзҡ„е…үзҡ„ж•Ҹж„ҹзЁӢеәҰ, зӣ®еүҚжүҖдҪҝз”Ёзҡ„е“Қеә”жӣІзәҝжҳҜеҹәдәҺе®ӨеҶ…зҡ„е®һйӘҢеҫ—еҲ°зҡ„, дәәзңјеңЁиҫғжҡ—зҺҜеўғеҜ№йўңиүІзҡ„ж•Ҹж„ҹеәҰдјҡйҷҚдҪҺ, еӣ жӯӨж— жі•еҫҲеҘҪзҡ„иЎЁзӨәе®ӨеӨ–е…үз…§зҺҜеўғ, дҪҶд»Қз„¶е°Ҷе…¶дҪңдёәз ”з©¶еҹәеҮҶ.

Y=вҲ«О»LО»(О»)V(О»)dО» \begin{equation} Y = \int_{\lambda} L_\lambda(\lambda) V(\lambda) \mathrm{d}\lambda \end{equation} Y=вҲ«О»вҖӢLО»вҖӢ(О»)V(О»)dО»вҖӢвҖӢ

иҫҗе°„еәҰйҮҸеҖјзҡ„з§ҜеҲҶ

Float PiecewiseLinearSpectrum::operator()(Float lambda) const {
    // Handle _PiecewiseLinearSpectrum_ corner cases
    if (lambdas.empty() || lambda < lambdas.front() || lambda > lambdas.back())
        return 0;

    // Find offset to largest _lambdas_ below _lambda_ and interpolate
    int o = FindInterval(lambdas.size(), [&](int i) { return lambdas[i] <= lambda; });
    DCHECK(lambda >= lambdas[o] && lambda <= lambdas[o + 1]);
    Float t = (lambda - lambdas[o]) / (lambdas[o + 1] - lambdas[o]);
    return Lerp(t, values[o], values[o + 1]);
}

static SampledWavelengths SampleUniform(Float u, Float lambda_min = Lambda_min,
                                        Float lambda_max = Lambda_max) {
    SampledWavelengths swl;
    // Sample first wavelength using _u_
    swl.lambda[0] = Lerp(u, lambda_min, lambda_max);

    // Initialize _lambda_ for remaining wavelengths
    Float delta = (lambda_max - lambda_min) / NSpectrumSamples;
    for (int i = 1; i < NSpectrumSamples; ++i) {
        swl.lambda[i] = swl.lambda[i - 1] + delta;
        if (swl.lambda[i] > lambda_max)
            swl.lambda[i] = lambda_min + (swl.lambda[i] - lambda_max);
    }

    // Compute PDF for sampled wavelengths
    for (int i = 0; i < NSpectrumSamples; ++i)
        swl.pdf[i] = 1 / (lambda_max - lambda_min);

    return swl;
}

XYZ SampledSpectrum::ToXYZ(const SampledWavelengths &lambda) const {
    // Sample the $X$, $Y$, and $Z$ matching curves at _lambda_
    SampledSpectrum X = Spectra::X().Sample(lambda);
    SampledSpectrum Y = Spectra::Y().Sample(lambda);
    SampledSpectrum Z = Spectra::Z().Sample(lambda);

    // Evaluate estimator to compute $(x,y,z)$ coefficients
    SampledSpectrum pdf = lambda.PDF();
    return XYZ(SafeDiv(X * *this, pdf).Average(), SafeDiv(Y * *this, pdf).Average(),
               SafeDiv(Z * *this, pdf).Average()) /
           CIE_Y_integral;
}

RGBUnboundedSpectrum::RGBUnboundedSpectrum(const RGBColorSpace &cs,
                                           RGB rgb) {
    Float m = std::max({rgb.r, rgb.g, rgb.b});
    scale = 2 * m;
    rsp = cs.ToRGBCoeffs(scale ? rgb / scale : RGB(0, 0, 0));
}
Float RGBUnboundedSpectrum::operator()(Float lambda) const { return scale * rsp(lambda); }
Float RGBUnboundedSpectrum::MaxValue() const { return scale * rsp.MaxValue(); }

RGBIlluminantSpectrum::RGBIlluminantSpectrum(const RGBColorSpace &cs,
                                             RGB rgb)
    : illuminant(&cs.illuminant) {
    Float m = std::max({rgb.r, rgb.g, rgb.b});
    scale = 2 * m;
    rsp = cs.ToRGBCoeffs(scale ? rgb / scale : RGB(0, 0, 0));
}
Float RGBIlluminantSpectrum::operator()(Float lambda) const {
    if (!illuminant) return 0;
    return scale * rsp(lambda) * (*illuminant)(lambda);
}
